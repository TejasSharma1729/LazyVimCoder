# LazyVimCoder

This is a fork of the LazyVim project with plugins installed including gemini-cli and vscode dark theme.
To install, run this command (requires Neovim version >= 0.11.2):
```bash
mkdir -p ~/.config/ # Ensure config directory exists
rm -rf ~/.config/nvim # Clear old configurations
git clone https://github.com/TejasSharma1729/LazyVimCoder.git ~/.config/nvim
nvim
```

LazyVim: here is a starter template for [LazyVim](https://github.com/LazyVim/LazyVim).
Refer to the [documentation](https://lazyvim.github.io/installation) to get started.
